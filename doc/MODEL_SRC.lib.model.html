MODEL_SRC.lib.model module¶

Collection of low dimensional system with known governing equations.

MODEL_SRC.lib.model.F_duffing_2d_system(x)[source]¶

Simple toy problem with multiple attractors in Otto’s paper.

Warning

Returned Jacobian is not used any more.

Note:

In this example, \(delta=0.5\), \(beta=-1.0\), \(alpha=1.0\).

Args:

x (numpy.ndarray): system state.

Returns:

tuple : A tuple of time derivative and Jacobian.

Simple toy problem with known Koopman eigenfunctions in Lusch paper.

Warning

Note:

In this example, \(\mu\) = -0.05, \(\lambda\) = -1.0.

Args:

x (numpy.ndarray): system state.

Returns:

tuple : A tuple of time derivative and Jacobian.
