class EVAL_SRC.lib.lib_model_interface.ClassFNN_edward(sess, layer_structure_list, act_fun_str, prefix, mode, LRAN_T, SVD_DMD, POD_W, nsd, W_b_K_dict)[source]¶

Bases: object

Class of Feedforward neural network given topology, structure, activation function, and recurrent or not, SVD_DMD or not, to replicate the model we saved.

Note:

We did this simply because there is no good save_model functionality in edward.
Args:

sess (class): the session we used in tensorflow.

layer_structure_list (list): the layer structure we used in constructing the Feedforward neural network.

act_fun_str (str): the name of the activation function to use.

prefix (str): the prefix used in the weights and biases parameters, either dec or enc.

mode (str): the type of ADVI to use, either MAPSimple or ADVInoARD, ADVIARD.

LRAN_T (int): the look forward time window length.

SVD_DMD (bool): flag on whether or not SVD DMD is enabled.

Attributes:

LRAN_T (int): the look forward time window length.

act_fun (function): the activation function chosen.

sess (class): the session we used in tensorflow.

prefix (str): the prefix used in the weights and biases parameters, either dec or enc.

mode (str): the type of ADVI to use, either MAPSimple or ADVInoARD, ADVIARD.

layer_structure_list (list): the layer structure we used in constructing the Feedforward neural network.

number_layer (int): the number of layers in the network.

graph (class): the graph assoicated with the session.

SVD_DMD (bool): flag on whether or not SVD DMD is enabled.

x (tf.Tensor): the placeholder for the input of FNN.

w_ph_list (list): a list containing the placeholders for weights. Since we treat all NN parameters
as placeholder, we naturally have a model to use and a NN constructed.

b_ph_list (list): a list containing the placeohlder for biases.

output (tf.Tensor): the tensor of the output of the FNN.

rv_w_list (list): a list of all probability distributions of weights in each layer.

rv_b_list (list): a list of all probability distributions of biases in each layer.

noise_lin (tf.Tensor): distribution of linear dynamics noises

noise_rec (tf.Tensor): distribution of reconstruction dynamics noises

K_K_X (tf.Tensor): the distribution of K_K_X (it is a normal with softplus scale)

K_SD (tf.Tensor): the distribution of K_SD (it is log normal)

num_samples (int): the total number of samples for Monte carlo sampling.

sample_rv_w_list (list): list of samples of weights in each layer

sample_rv_b_list (list): list of samples of biases in each layer

sample_K (numpy.ndarray): array of realization of K matrix.

sample_lambda_lin (numpy.ndarray): array of realization of scale of linear dynamics noises.

sample_lambda_rec (numpy.ndarray): array of realization of scale of reconstruction noises.

sample_noise_lin (numpy.ndarray): array of realization of linear dynamics noises.

sample_noise_rec (numpy.ndarray): array of realization of reconstruction noises.
