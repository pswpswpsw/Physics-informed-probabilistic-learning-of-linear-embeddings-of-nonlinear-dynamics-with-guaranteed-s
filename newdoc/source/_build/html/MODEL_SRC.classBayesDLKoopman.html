Bases: object

Class of Deep Leraning Koopman with Bayesian Deep Learning.

Args:
data (numpy.ndarray): training data with key Xtrain and XdotTrain for LRAN_mode =
False while only with Xtrain otherwise.

configDict (dict): config dictionary for deterministic DL Koopman

mc_sample (int): number of Monte carlo sampling in using the reparameterization trick.
We recommend 1. The higher it is, the less variance there is in the gradient estimation.

edward_configDict (dict): config dictionary for bayesian DL Koopman.

cut_data_range (numpy.ndarray): Specifiy the range for training data. If None, then we don’t use it.

LRAN_mode (bool): Using LRAN or not.

gpu_percentage (float): gpu memory usage ratio. Default value is 0.5.

Attributes:

model (class): Deterministic DL Koopman model class from ClassDLKoopmanCont.

residual_vector_rec_loss (tf.Tensor): residual reconstruction loss vector as the output of deterministic
network as likelihood.
residual_vector_lin_loss (tf.Tensor): residual linear loss vector as the output of deterministic
network as likelihood.

z_ph (tf.Tensor): concanteted z_rec_loss and z_lin_loss as general loss target to minimize.

vp_dict (dict): dictionary collection of variational posteriori

sess (class): current session used in Tensorflow.

graph (class): current graph being used.

verbose (bool): logging flags.

mode (str): choose which mode to take on ADVI.

LRAN_mode (bool): choose using recurrent or differential form.

dir (str): path to saving directory.

mc_sample (int): number of MC samples to choose for approximating the reparameterization trick.

init_mode (str): Standard or nonBayes as using default weights initialization or simply use previous run.

MAP_dir (str): if init_mode is turned with nonBayes, then one needs to give the path to previous trained model.

init_std_softplus (float): initialization of the standard deviation for softplus for the scales. The smaller the more negative, the weights will be smaller.

optimizer (str): the name for the optimizer in tensorflow used in the training process. For example, adam.
